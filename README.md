📖 PROJECT MANIFESTO
ScriptTelegramNewCpanel adalah ekosistem otomatisasi Telegram yang dirancang untuk kecepatan, efisiensi, dan stabilitas absolut. Proyek ini merupakan manifestasi kolaborasi antara kedalaman teknis Kelvin Cyber dalam infrastruktur dan presisi analitis dari Gemini AI. Kami tidak hanya menulis kode; kami merancang entitas digital yang tangguh.
"Speed is the new currency. Stability is the new standard."
🚀 WHY DIRECT EXECUTION?
⚡ Zero Latency: Bot tidak perlu melakukan handshake ke server luar saat booting.
🛡️ Maximum Stability: Menghilangkan risiko 404 Not Found atau Timeout saat pengambilan token.
⚙️ High Performance: Memangkas penggunaan memori server karena tidak perlu menjalankan fungsi request dan decryption yang berat.
🔌 Ready-to-Deploy: Sangat cocok untuk lingkungan hosting (Pterodactyl, VPS, Cloud) dengan batasan akses internet keluar.

📂 FOLDER STRUCTURE
```text
/ScriptTelegramNewCpanel
│
├── main.py          # Entry point utama (Engine)
├── config.py        # Pengaturan token dan API
├── keep_alive.py    # Menjaga bot tetap aktif 24/7
├── /handlers        # Modul fungsionalitas
│   ├── start.py     # Handling command /start
│   ├── admin.py     # Otoritas akses admin
│   ├── create.py    # Logika pembuatan data
│   └── utils.py     # Utility tools tambahan
└── requirements.txt # Daftar dependensi sistem
```
🛠 TECHNICAL SPECIFICATIONS
Language: Python 3.11+
Core Engine: pyTelegramBotAPI
Deployment Mode: Direct Execution
Architecture: Lightweight & Fast Modular Design

🔐 SECURITY & STABILITY
Bot ini menggunakan prinsip Separation of Concerns. Logika bot dipisahkan dalam modul-modul kecil agar jika terjadi bug pada satu fitur, fitur lainnya tetap berjalan normal. Ini memastikan uptime bot mencapai titik maksimal.

🚀 QUICK START
Clone repository ini.
Edit file config.py dan masukkan token Telegram kamu di variabel BOT_TOKEN.
Install dependensi: pip install -r requirements.txt
Jalankan bot: python main.py
Bot siap melayani permintaan!
