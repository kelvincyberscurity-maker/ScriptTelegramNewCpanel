# 📖 Project Overview

**ScriptTelegramNewCpanel** merupakan bot Telegram berbasis Python yang dirancang dengan fokus pada **kecepatan**, **stabilitas**, dan **arsitektur modular**.

Proyek ini menggabungkan pengalaman teknis **Kelvin Cyber** dalam membangun infrastruktur dengan bantuan analisis dari **Gemini AI**, sehingga menghasilkan sistem yang ringan, mudah dikembangkan, dan siap digunakan pada berbagai lingkungan server.

> **"Speed is the new currency. Stability is the new standard."**

---

# 🚀 Why Direct Execution?

Direct Execution memungkinkan bot berjalan tanpa mengambil konfigurasi dari server eksternal saat proses startup.

### ⚡ Zero Latency
Tidak memerlukan request tambahan saat proses booting sehingga startup menjadi lebih cepat.

### 🛡 Maximum Stability
Mengurangi risiko error seperti **Timeout**, **404 Not Found**, maupun kegagalan request jaringan.

### ⚙ High Performance
Menghemat penggunaan CPU dan RAM karena tidak menjalankan proses decrypt maupun fetch konfigurasi.

### 🔌 Ready for Deployment
Optimal untuk dijalankan pada:

- Pterodactyl Panel
- VPS
- Dedicated Server
- Cloud Server

# 🔐 Security & Stability

Bot dikembangkan menggunakan prinsip **Separation of Concerns**, yaitu setiap fitur dipisahkan ke dalam modul tersendiri.

Keuntungan pendekatan ini:

- Modular dan mudah dikembangkan
- Mudah melakukan maintenance
- Risiko bug lebih kecil
- Performa lebih stabil
- Uptime lebih maksimal

# ⚡ Quick Start

### 1. Clone Repository

```bash
git clone https://github.com/username/ScriptTelegramNewCpanel.git
```

### 2. Masuk ke Folder

```bash
cd ScriptTelegramNewCpanel
```

### 3. Install Dependency

```bash
pip install -r requirements.txt
```

### 4. Konfigurasi

Buka file:

```text
config.py
```

Masukkan token Telegram:

```python
BOT_TOKEN = "YOUR_TELEGRAM_BOT_TOKEN"
```

### 5. Jalankan Bot

```bash
python main.py
```

Bot siap digunakan.
